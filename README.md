# Climate Analyser 🌤️
A Python desktop app using **OpenWeatherMap API**.

### Key Features:
- **Multi-threading:** No UI freezing while loading.
- **Smart Tips:** Health advice based on Pune's heat (38°C+).
- **Security:** Managed via `.env` variables.

### How to run:
1. Install `pip install -r requirements.txt`
2. Add your API Key to a `.env` file.
3. Run `python weatherAnylyser.py`
